# Requires these packages installed through Terminal (requires Homebrew)

Babel JavaScript compiler

`npm i --save-dev
@babel/core babel-loader
@babel/preset-env
@babel/preset-react
@babel/preset-typescript`

Webpack asset bundler

`npm i --save-dev
webpack
webpack-cli
webpack-dev-server
fork-ts-checker-webpack-plugin
html-webpack-plugin`

React.js itself

`npm i
react
react-dom`

React Refresh for hot reloading

`npm i --save-dev
react-refresh
@pmmmwh/react-refresh-webpack-plugin`

ReactDOM for DOM-specific methods

`npm i --save-dev @types/react-dom`

And finally, TypeScript!

`npm i --save-dev typescript`

# How to run

1. On Terminal, set cd to the project's working directory
2. Run: `npm run start`
3. Control+C to stop run
